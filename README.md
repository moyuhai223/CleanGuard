# CleanGuard

生产部劳保与更衣柜管理系统（CG-200）初始开发骨架，基于 .NET Framework 4.6.2 + SQLite。

## 当前已实现

- WinForms 项目骨架（主界面、编辑界面）
- SQLite 自动初始化（按 V4.0 文档建表）
- 默认柜位字典自动初始化（1F/2F，各 60 个衣柜 + 60 个鞋柜）
- 员工列表查询（姓名/拼音/工序模糊检索）
- 新增员工保存（含工号唯一、柜位类型与占用校验）
- 四柜位下拉筛选（1F/2F + 衣柜/鞋柜）
- 离职办理（释放四柜位资源并写系统日志）
- 批量导入向导（CSV模板下载 + CSV导入校验 + 失败明细导出 + 导入日志）
- 打印标签（打印预览 + 二维码内容日志）
- 柜位分布图入口（当前为统计弹窗版）
- 程序退出自动备份数据库并清理 7 天前备份

## 目录

- `Src/CleanGuard_App`：主程序代码
- `Src/Database/init.sql`：数据库初始化脚本
- `Doc`：文档（预留）
